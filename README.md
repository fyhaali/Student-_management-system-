# نظام إدارة الطلاب المتكامل (النسخة الثانية)

مشروع تطوير نظام تسجيل طلاب ديناميكي وآمن باستخدام لغة **PHP** وقاعدة بيانات **MySQL** والاتصال عبر تقنية **PDO**.

## 🚀 مميزات المشروع
* **اتصال آمن ببيئة PDO:** تم بناء نظام اتصال مرن وآمن يمنع تسريب الأخطاء الحرجة.
* **الاستعلامات المجهزة (Prepared Statements):** حماية كاملة للبيانات المدخلة والمحذوفة ضد هجمات حجب الخدمة واختراق الثغرات (SQL Injection).
* **تنظيف وتحقق المدخلات (Data Validation):** فلترة البريد الإلكتروني والتأكد من الحقول الفريدة لضمان سلامة قاعدة البيانات.
* **إدارة كاملة (CRUD - Create & Delete):** واجهة متكاملة لإضافة الطلاب، وعرضهم في جدول HTML، وحذف أي سجل بضغطة زر وتحديث تلقائي للقائمة.

---

## 🛠️ متطلبات التشغيل المحلي
1. سيرفر محلي مثل **XAMPP** أو **WampServer**.
2. تفعيل خدمة **Apache** و **MySQL**.

---

## 💻 خطوات الإعداد والتشغيل

1. **إعداد قاعدة البيانات:**
   * افتح `phpMyAdmin`.
   * قم بإنشاء قاعدة بيانات باسم: `student_management`.
   * اذهب إلى تبويب **SQL** واسترِد أو انسخ محتوى الملف المرفق في المستودع باسم `قاعدة البيانات.SQL` لتوليد الجدول فوراً.

2. **تجهيز الكود:**
   * ضع مجلد المشروع داخل مسار السيرفر المحلي (مثلاً `htdocs` في سيرفر XAMPP).
   * تأكد من إعدادات الاتصال بداخل ملف `do.php` (اسم المستخدم الافتراضي `root` وبدون كلمة مرور).

3. **تشغيل النظام:**
   * افتح المتصفح واذهب إلى: `http://localhost/اسم_مجلد_المشروع/index.php`.
